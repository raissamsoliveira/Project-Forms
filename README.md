# Formulário de Cadastro para Analista de Experiência do Usuário (CX)

Este projeto visa criar um formulário de cadastro para candidatos interessados em vagas de Analistas de Experiência do Usuário na empresa RAJO. O formulário coleta informações essenciais, como nome, WhatsApp, cidade de residência e nível de conhecimento em áreas como Customer Success (CS), Customer Experience (CX) e atendimento ao cliente. O objetivo é fornecer uma interface intuitiva para facilitar o processo de inscrição e captura de dados relevantes para a seleção de candidatos qualificados.

## Estrutura do Repositório

- `/assets` - Contém os arquivos de estilo CSS utilizados no formulário.
- `index.html` - Estrutura principal do formulário em HTML.
- `README.md` - Documentação do projeto, incluindo descrição, instruções de uso e detalhes técnicos.

## Funcionalidades

1. **Coleta de Dados Pessoais**: Nome completo, WhatsApp, e cidade de residência.
2. **Disponibilidade para Trabalho**: Inclui opções de disponibilidade para trabalho remoto e início imediato.
3. **Avaliação de Competências**: Permite aos candidatos avaliarem seu nível de conhecimento em CS, CX, atendimento ao cliente e outras competências.
4. **Experiência Anterior e Motivação**: Campos para descrever experiências anteriores relevantes e o motivo de interesse em trabalhar na RAJO.
5. **Upload de Currículo**: Opção para o envio do currículo em formatos como PDF ou DOCX.

## Tecnologias Utilizadas

- **HTML:** Para a estruturação do formulário.
- **CSS:** Para o design e responsividade da página.
- **JavaScript:** (se aplicável) para validação e funcionalidades interativas.

## Como Utilizar o Projeto

1. Clone o repositório:

   ```bash
   git clone https://github.com/SeuUsuario/formulario-cadastro-rajo.git
   ```

2. Navegue até o diretório do projeto: 

    ```bash
    cd formulario-cadastro-rajo
   ```
3. Abra o arquivo index.html no seu navegador para visualizar o formulário.
4. Personalize o formulário conforme necessário, ajustando as perguntas ou o estilo no arquivo CSS.

## Exemplos e Personalizações
Este projeto é altamente personalizável. Adapte-o conforme as necessidades da sua empresa ou para diferentes posições, alterando os campos e estilos no arquivo HTML e CSS.
